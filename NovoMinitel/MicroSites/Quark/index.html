Quark Interactive Designer
O Quark Interactive Designer é a única ferramenta que oferece simultaneamente design para impressão, HTML e Flash, tudo numa aplicação e recorrendo a técnicas simples e intuitivas para adicionar som, movimento e interactividade aos objectos de uma maqueta.
 
 
QuarkXPress 7.3

Tenha o controlo da transparência com toda a liberdade. Crie gráficos visualmente fascinantes dentro do próprio QuarkXPress 7.3 - sem necessitar de recorrer a aplicações externas. Trabalhe de forma mais eficiente, graças a um interface moderno, que integra um conjunto de ferramentas de desenho modernas.
 
 
Quark Print Collection
Um interface intuitivo permite-lhe impor a partir do QuarkXPress 7 layouts ou ficheiros PDF para uma impressora pronta a imprimir com a capacidade de poder fazer uma selecção de diferentes tipos de papel e diferentes estilos de paginação.
 
 
Atomik XML Publisher
Para os milhares de empresas que utilizam o QuarkXPress, na criação de conteúdos impressos de alta qualidade, a Quark passou a distribuir o Atomik XML Publisher, de forma a disponibilizar mais rapidamente os conteúdos para Internet e dispositivos móveis.
