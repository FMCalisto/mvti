Para os milhares de empresas que utilizam o QuarkXPress, na criação de conteúdos impressos de alta qualidade, a Quark passou a distribuir o Atomik XML Publisher, de forma a disponibilizar mais rapidamente os conteúdos para Internet e dispositivos móveis.

Através da linguagem XML, o processo de exportar para websites e de importar conteúdos da Internet para desenvolver trabalhos impressos, é agora muito mais fácil e rentável, maximizando o trabalho de uma forma estruturada.

Funcionalidades:

Extrai automaticamente os conteúdos para QuarkXPress em formato XML
O Atomik Xport permite-lhe extrair os conteúdos de documentos QuarkXPress em XML estruturado e valido.

Processamento documento a documento
O sistema de correspondência automatizado do Atomik Xport permite-lhe a extracção automática da totalidade dos documentos do QuarkXPress em formato XML

Extracção automática das imagens
O Atomik Xport extrai automaticamente as referências a imagens. Isto inclui o nome da imagem, a localização bem como outros atributos, mantendo a relação entre o texto e a imagem.

Compatível com parágrafos e estilos de letra.
O Atomik Xport consegue identificar os conteúdos analisando a formatação dos parágrafos, bem como de letra. Isto permite uma extracção de conteúdos bastante fiel.

Funciona com tabulações
Utilizando avançadas técnicas, o Atomik Xport mantém as colunas e as linhas das tabelas, permitindo aos dados serem extraídos em tabelas formatadas no formato XML.

Verificação do estrutura XML do documento.
Cria ficheiros XML válidos e bem formatos que cumpre os padrões predefinidos DTD e XML.

Pré-visualização dos conteúdos:
Visualize os conteúdos no Atomik Xport, antes do XML ser criado ou configure o Atomik Xport para criar ficheiros dos conteúdos ignorados.

Funciona com  Quark Publishing System™ (QPS™)
Compatível com documentos QuarkXPress armazenados em QPS. A automatização automática é possível se utilizar o Atomik Xport Developer Edition.


Automatização em ambiente Server.
O Atomik Xport Developer edition (disponível em separado) permite automatização, processando os documentos do QuarkXPress em formato XML para um ambiente server.
