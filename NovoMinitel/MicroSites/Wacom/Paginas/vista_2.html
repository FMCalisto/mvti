Microsoft Windows Vista - Reconhecimento de escrita
 
 

O Windows vista introduz o reconhecimento inteligente de escrita manual, como um complemento dos dados introduzidos no teclado.

Esta nova tecnologia de reconhecimento de escrita, eficiente e precisa, converte uma grande variedade de textos manuscritos em textos dactilografados. Os textos manuscritos são uma forma rápida de introduzir informações, em qualquer zona que o computador aceite a introdução de dados.
   

São muitas as circunstâncias em que é bastante útil possuir esta opção para escrever com uma caneta, como complemento dos dados introduzidos pelo teclado.

O Vista disponibiliza esta opção de uma forma bastante aprimorada. Os dados introduzidos pela caneta são capturados como se tratasse de tinta, sendo reconhecidos no momento de uma forma bastante precisa, que torna os dados instantaneamente pesquisáveis.

A escrita manual pode ser convertida em qualquer momento para texto. Escrever com uma caneta, é uma forma particularmente eficiente e natural de comentar um documento existente, ou de tirar apontamentos de pensamentos soltos que vão surgindo.
